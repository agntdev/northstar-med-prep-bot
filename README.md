# Northstar Med Prep Bot

Telegram bot for Northstar Med Prep to browse courses, enquire, manage student reminders, quizzes and doubt tickets with human-in-the-loop staff notifications.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
